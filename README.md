## 📖 About

This is the source code of the personal developer portfolio. The developer is self-taught and is based out of Karachi, Pakistan. He/she specializes in developing PHP applications and customized WordPress/WooCommerce websites. The website has been designed to showcase various working projects, such as the role-based PHP system and other customized e-commerce sites.

All this can be achieved using only hand-written coding into the index.html file, in order to keep the site lightweight and dependency free.

## 🚀 Getting Started

Nothing needs to be installed or configured. | **Section** | 

```bash
# 1. Clone the repository
git clone https://github.com/dawoodhabib46/Dawood-Habib-Portfolio.git

# 2. Move into the project folder
cd Dawood-Habib-Portfolio

# 3. Open it directly in your browser
open index.html

# — or serve it locally for the best experience —
# using VS Code's "Live Server" extension, or:
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
